# Pinterest Clone

## Project Description
This is a Pinterest clone web application that allows users to sign up, log in, upload posts with images, view profiles, and interact similarly to Pinterest. It features user authentication, image uploads, and a user-friendly interface.

## Features
- User registration and login using Passport.js authentication
- Profile management
- Upload posts with image support via Multer
- View posts by various users
- Responsive design and smooth user experience
- Flash messages for confirmation and error notifications

## Technologies Used
- Node.js
- Express.js
- MongoDB with Mongoose
- Passport.js for authentication
- Multer for file uploads
- Flash-connect for flash messages
- EJS or your preferred templating engine (specify if different)

## Installation
1. Clone the repository:
   git clone
2. Navigate to the project directory:
   cd pinterest-clone
3.install dependencies:
  npm install
4. Set up environment variables:
  PORT=3000
   MONGODB_URI=mongodb://localhost:27017/pinterest-clone
    SESSION_SECRET=yourSessionSecret
5.start server
   npm start

## Usage
- Open your browser and go to http://localhost:3000
- Register a new account or log in if you already have one
- Upload posts with images and manage your profile
- Browse other users' posts
